Система CMS блога на PHP

Данный шаблон серьезнее предыдущего. Сайт также был создан для какой-то проектной работы в колледже. На данном сайте присутствуют последние публикации, список категорий, поиск по сайту а также секция слайдер с топ публикациями. Здесь разработан модуль авторизации пользователя на сайт, через базу данных MySql выдаются права пользователю для пользования сайта.
